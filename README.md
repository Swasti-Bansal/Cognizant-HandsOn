# Cognizant — Hands-On Training

## ▶️ How to Run

### Java Exercises

Navigate to the required exercise folder:

```bash
cd "<exercise-folder>"
```

Compile:

```bash
javac *.java
```

Run:

```bash
java <MainClassName>
```

Examples:

```bash
java SingletonTest
java FactoryTest
java SearchTest
java ForecastTest
```

### PL/SQL Exercises

Execute the `.sql` files using Oracle SQL Developer, SQL*Plus, or Oracle Live SQL.

### JUnit Exercises

JUnit exercises require JUnit 4.13.2 and can be executed using Maven or an IDE with JUnit support.
